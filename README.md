# ToolGPT
A ai-tool made by ChatGPT and Langchain with Streamlit UI

## Requirements
### Packages
- langchain
- streamlit
- duckduckgo-search

### Install dependence with
```
pip install -r requirement.txt
```

## Run
- Install dependences
- Edit the config.py
- Run webui:
```
streamlit run webui.py
```
